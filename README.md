# Tweeter Project

Tweeter is a simple, single-page Twitter clone.

Starter code cloned from Lighthouse Labs repo. Developed further using HTML, CSS, JS, jQuery and AJAX.

## Dependencies

- Express
- Node 5.10.x or above
- Body parser
- Chance
- md5
